# bright

Adjust screen's brightness using [brightnessctl](https://github.com/Hummer12007/brightnessctl).

![view of block](block.jpg)

bright is a minimal useful brightness control for i3blocks.

# Config

```ini
[bright]
color=#d8dee9
background=#00000000
command=$SCRIPT_DIR/bright
LABEL=ï††
interval=1
signal=10
min_width=26
align=center
```
